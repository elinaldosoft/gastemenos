### How to install the project
- git clone git@github.com:your-username-here/gastemenos.git gastemenos
- cd gastemenos
- python3.11 -m venv venv
- source venv/bin/activate
- poetry install
- python manage.py check

### Running project
- python manage.py runserver
- http://127.0.0.1:8000/
